Sayembara Untuk Mahasiswa Mata Kuliah Keamanan Informasi

Sayembara Apakah itu?

Keamanan informasi merupakan salah satu mata kuliah yang sangat menyenangkan, karena ruang lingkup bidang keamanan informasi sendiri sangat luas. Oleh karena itu, saya tertarik untuk membuat sayembara terkait mata kuliah ini, khususnya pada materi Kriptografi agar dikerjakan oleh mahasiswa yang mengambil mata kuliah tersebut.

Sayembara akan terus dibuka sampai ada mahasiswa yang berhasil memecahkan soal yang ada pada sayembara ini. Soal sayembaranya adalah :

Soal Sayembara

Adapun kode program vigenere ciphertextnya adalah sebagai berikut : 

#include 
#define KEY_LENGTH 2

main(){
    unsigned char ch;
    FILE *fpIn, *fpOut;
    int i;
    unsigned char key[KEY_LENGTH] = {0x00, 0x00};


    fpIn = fopen("ptext.txt", "r");
    fpOut = fopen("ctext.txt", "w");

    i=0;
    while (fscanf(fpIn, "%c", &ch) != EOF) {

      if (ch!='\n') {
        fprintf(fpOut, "%02X", ch ^ key[i % KEY_LENGTH]);
        i++;
      }
    }

    fclose(fpIn);
    fclose(fpOut);
    return;
}

Kode program diatas dibuat menggunakan bahasa C. Dengan menggunakan kode program dan ciphertext tersebut, silakan carilah nama "Plaintext" nya.
